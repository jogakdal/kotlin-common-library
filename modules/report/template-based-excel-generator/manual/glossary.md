# TBEG 용어집

TBEG 문서에서 사용하는 주요 용어를 정리합니다.

---

## 마커 (Marker)

Excel 템플릿 셀에 배치하는 특수 표현식입니다. TBEG은 마커를 실제 데이터로 치환하여 Excel 파일을 생성합니다.

- **변수형 마커**: `${변수명}` 형태. 단순 변수 치환에 사용합니다. 예: `${title}`, `${emp.name}`
- **함수형 마커**: `${함수(파라미터)}` 형태. repeat, image, merge 등 특수 기능을 수행합니다. 예: `${repeat(employees, A2:C2, emp)}`

## 반복 영역 (Repeat Region)

`${repeat(...)}` 마커가 지정한 셀 범위입니다. 컬렉션의 각 아이템마다 이 범위가 복제되어 확장됩니다. 기본적으로 아래 방향(DOWN)으로 확장되며, 오른쪽 방향(RIGHT)도 지원합니다.

## 번들 (Bundle)

`${bundle(범위)}` 마커로 지정하는 셀 범위 묶음입니다. 번들 안의 요소는 하나의 단위로 취급되어, 다른 반복 영역의 확장에 영향받지 않고 독립적으로 동작합니다. 주로 같은 시트에 여러 반복 영역이 있을 때 레이아웃을 보호하는 용도로 사용합니다.

## DataProvider

Excel 템플릿에 바인딩할 데이터를 제공하는 인터페이스(`ExcelDataProvider`)입니다. 단순한 `Map<String, Any>` 대신 사용하면 지연 로딩, 대용량 스트리밍 등 고급 데이터 제공 전략을 구현할 수 있습니다.

- **SimpleDataProvider**: `ExcelDataProvider`의 기본 구현체. DSL 또는 Builder 패턴으로 간편하게 생성합니다.
- **커스텀 DataProvider**: `ExcelDataProvider` 인터페이스를 직접 구현하여 DB 스트리밍, 외부 API 호출 등 특수한 데이터 소스를 연동합니다.

## 선택적 필드 노출 (Hideable)

상황에 따라 특정 필드의 노출을 제한하는 기능입니다. `${hideable(...)}` 마커를 템플릿에 배치하고, 코드에서 `hideFields()`로 숨길 필드를 지정합니다.

- **DELETE 모드**: 해당 영역을 물리적으로 삭제하고 나머지 요소를 당깁니다 (기본값).
- **DIM 모드**: 데이터 영역에 비활성화 스타일(회색 배경 + 연한 글자색)을 적용하고 값을 제거합니다. 필드 타이틀 등 repeat 밖 bundle 영역은 글자색만 변경됩니다. 레이아웃은 유지됩니다.

## 셀 병합 (Merge)

`${merge(item.필드)}` 마커를 사용하여, 반복 확장 후 연속된 같은 값의 셀을 자동으로 병합하는 기능입니다. 병합이 올바르게 동작하려면 해당 필드 기준으로 데이터를 미리 정렬해야 합니다.

## 수식 바인딩 (Formula Binding)

Excel 수식 셀에 `${변수명}` 마커를 배치하여, 수식의 일부를 동적으로 치환하는 기능입니다. 예를 들어 `=HYPERLINK(${emp.url}, ${emp.name})`처럼 수식 인자에 변수를 바인딩할 수 있습니다.

## 수식 자동 조정 (Formula Adjustment)

반복 영역이 확장될 때, 영향받는 수식의 셀 참조 범위를 자동으로 갱신하는 기능입니다. 예를 들어 `=AVERAGE(B2:B2)`가 100개 행으로 확장되면 `=AVERAGE(B2:B101)`로 자동 조정됩니다. `SUM`, `AVERAGE`, `COUNT`, `MAX` 등 범위를 참조하는 모든 수식에 적용됩니다.

## 지연 로딩 (Lazy Loading)

DataProvider에서 데이터를 미리 메모리에 로드하지 않고, TBEG이 실제로 데이터를 필요로 하는 시점에 Lambda/Supplier를 호출하여 로드하는 방식입니다. 대용량 데이터 처리 시 메모리 사용량을 크게 줄일 수 있습니다.

## count

컬렉션의 전체 아이템 수입니다. DataProvider에서 count를 미리 제공하면 TBEG이 데이터를 이중 순회하지 않아 성능이 향상됩니다. count를 제공하지 않으면 TBEG이 데이터를 먼저 순회하여 개수를 파악합니다.

## 템플릿 (Template)

TBEG이 데이터를 바인딩할 원본 Excel 파일(.xlsx)입니다. 서식, 수식, 조건부 서식, 차트 등 Excel 네이티브 기능을 자유롭게 사용하되, 데이터가 들어갈 위치에 마커를 배치합니다.
